USER TESTING AND OUTCOMES

User Tests

After some user tests were conducted I got some valuable feedback to how users were feeling about the lo-fidelity website.

1. Put cart all the way to the right of the page.

2. Add a number to the cart to indicate how many items have been added.

3. Have less text on the website. It was feeling overwhelming to users.

4. Add best sellers and new arrivals.

5. Be able to click on "shop" and not just the dropdown menu.

6. Get rid of FAQ, Users didn't interact or care about it.

7. Be able to revisit items from cart to determine if the user actually wants the item.

All the above feedback and observations were implemented in the high-fidelity prototype.
